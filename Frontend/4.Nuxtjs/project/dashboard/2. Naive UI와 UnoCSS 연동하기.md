# Nuxt 대시보드 프로젝트 시리즈 (2): Naive UI와 UnoCSS 연동하기

대시보드는 정보가 명확하게 전달되어야 합니다. Naive UI의 강력한 컴포넌트와 UnoCSS의 빠른 스타일링을 결합하여 디자인 시스템을 구축해 보겠습니다.

---

## 1. UnoCSS 설정 (uno.config.ts)

프로젝트 루트에 `uno.config.ts` 파일을 생성하여 기본 프리셋과 단축어(Shortcuts)를 정의합니다.

```typescript
// uno.config.ts
import { defineConfig, presetUno, presetAttributify, presetIcons } from 'unocss'

export default defineConfig({
  presets: [
    presetUno(), // 기본 프리셋
    presetAttributify(), // 속성 기반 스타일 지원
    presetIcons(), // 아이콘 사용 지원
  ],
  shortcuts: {
    'flex-center': 'flex items-center justify-center',
    'card-base': 'bg-white rounded-lg shadow-sm border border-gray-200 p-4',
  }
})
```

---

## 2. Naive UI 테마 커스터마이징

Naive UI는 자바스크립트 객체만으로 전체 테마를 쉽게 바꿀 수 있습니다.

```vue
<!-- app.vue -->
<script setup>
import { NConfigProvider, zhCN, dateZhCN } from 'naive-ui'

/** @type {import('naive-ui').GlobalThemeOverrides} */
const themeOverrides = {
  common: {
    primaryColor: '#18a058',
    primaryColorHover: '#36ad6a',
  },
  Button: {
    fontWeight: 'bold'
  }
}
</script>

<template>
  <n-config-provider :theme-overrides="themeOverrides">
    <NuxtLayout>
      <NuxtPage />
    </NuxtLayout>
  </n-config-provider>
</template>
```

## 3. 아이콘 시스템 구축

대시보드에서 아이콘은 필수입니다. UnoCSS의 `presetIcons`를 활용하면 별도의 SVG 파일 없이 클래스명만으로 수만 개의 아이콘을 쓸 수 있습니다.

```html
<!-- 사용 예시 -->
<div class="i-carbon-dashboard text-xl" /> <!-- 대시보드 아이콘 -->
<div class="i-carbon-user-avatar text-blue-500" /> <!-- 유저 아이콘 -->
```

---

## 4. 왜 UnoCSS인가?

*   **성능:** 사용하지 않는 클래스는 빌드 시 완전히 제거되어 CSS 용량이 극단적으로 작아집니다.
*   **유연성:** `presetAttributify`를 쓰면 `<div m-2 p-4>` 처럼 속성 형태로 코드를 작성할 수 있어 가독성이 좋아집니다.
