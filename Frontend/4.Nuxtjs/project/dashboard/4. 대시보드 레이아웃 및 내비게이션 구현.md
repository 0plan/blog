# Nuxt 대시보드 프로젝트 시리즈 (4): 대시보드 레이아웃 구현

이제 디자인 시스템과 상태 관리가 준비되었으니, 본격적으로 대시보드의 뼈대인 **레이아웃(Layout)**을 만들어 보겠습니다. Naive UI의 레이아웃 컴포넌트를 활용합니다.

---

## 1. 기본 레이아웃 구조 (layouts/default.vue)

사이드바, 헤더, 콘텐츠 영역으로 나뉜 표준 대시보드 구조입니다.

```vue
<template>
  <n-layout has-sider h-screen>
    <!-- 사이드바 -->
    <n-layout-sider
      bordered
      collapse-mode="width"
      :collapsed-width="64"
      :width="240"
      :collapsed="appStore.isSidebarCollapsed"
      show-trigger
      @collapse="appStore.toggleSidebar"
      @expand="appStore.toggleSidebar"
    >
      <div class="h-16 flex-center font-bold text-lg overflow-hidden whitespace-nowrap">
        {{ appStore.isSidebarCollapsed ? 'D' : 'Dashboard' }}
      </div>
      <n-menu :options="menuOptions" />
    </n-layout-sider>

    <n-layout>
      <!-- 헤더 -->
      <n-layout-header bordered class="h-16 flex items-center justify-between px-6">
        <div class="i-carbon-menu text-xl cursor-pointer" @click="appStore.toggleSidebar" />
        <div class="flex items-center gap-4">
          <span>{{ authStore.user.name }}</span>
          <n-avatar round :src="authStore.user.avatar" />
        </div>
      </n-layout-header>

      <!-- 콘텐츠 영역 -->
      <n-layout-content content-style="padding: 24px;" class="bg-gray-50/50">
        <slot />
      </n-layout-content>
    </n-layout>
  </n-layout>
</template>

<script setup>
import { useAppStore } from '@/stores/app'
import { useAuthStore } from '@/stores/auth'

const appStore = useAppStore()
const authStore = useAuthStore()

const menuOptions = [
  { label: '홈', key: 'home', icon: () => h('div', { class: 'i-carbon-home' }) },
  { label: '통계', key: 'stats', icon: () => h('div', { class: 'i-carbon-chart-line' }) },
  { label: '설정', key: 'settings', icon: () => h('div', { class: 'i-carbon-settings' }) },
]
</script>
```

---

## 2. 레이아웃의 핵심 포인트

1.  **`has-sider`:** `n-layout` 안에 사이드바가 포함될 때 반드시 추가해야 하는 속성입니다.
2.  **`slot`:** 각 페이지의 내용이 들어갈 자리입니다.
3.  **반응형 대응:** 사이드바 접힘 상태를 Pinia와 연동하여 헤더의 버튼이나 사이드바 자체 트리거로 제어할 수 있습니다.

---

## 3. UnoCSS로 여백과 배경 처리

`content-style` 외에도 UnoCSS를 사용하여 디테일한 스타일을 잡습니다.
*   `h-screen`: 전체 화면 높이 차지.
*   `bg-gray-50/50`: 콘텐츠 영역에 살짝 회색빛 배경을 주어 카드가 돋보이게 함.
